## TDE1 - Projeto de Operações com Conjuntos em Python
Aluno: Rafael Felipe Xavier da Silva

Curso: 6º Período BSI

Matéria: Resolução de Problemas de Natureza Discreta


Este projeto Python lê uma entrada a partir de um arquivo `.txt` e executa operações de conjunto sobre os dados. As operações suportadas são:

- **União**
- **Interseção**
- **Diferença**
- **Produto Cartesiano**

Ao executar o projeto, você deve fornecer o nome do arquivo de entrada que será processado.
